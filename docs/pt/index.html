Introdução¶

Esta aplicação é uma ferramenta simples para os utilizadores das bolsas de apostas e propõem-se a resolver o problema bastante conhecido de Bet-Hedging: equilibrar os lucros de duas apostas – uma a favor e outra contra – no mesmo acontecimento, ao serem realizadas em momentos diferentes.

O objectivo principal no seu desenvolvimento foi garantir a possibilidade de utilizar somente o teclado de forma rápida e eficiente. Obviamente, existem outras soluções do mesmo género [1] [2] [3] mas todas têm problemas: dependem do web-browser, nenhuma permite a devoção ao teclado, entre outros.

Esta calculado é um projecto open-source (GPLv2) e multi-plataforma. Foi implementada enquanto software livre com o objectivo de ser executada em todas as plataformas abrangidas pela biblioteca wxPython [4].

[1]http://www.arbcruncher.com/
[2]http://www.betcalc.com/backlaycalc.php
[3]http://www.oddschecker.com/betting-tools/hedging-calculator.htm
[4]http://www.wxpython.org/

Desenvolvimento¶

Sugestões, comentários ou reviews são bem-vindas! Podem enviar-me os mesmos/as para filipe.funenga@ist.utl.pt. Caso encontrem algum problema, podem criar um issue no github [5] ou enviar-me um email.

O desenvolvimento desta aplicação foi iniciado em 23/Dez/2012 e tem sido realizado no Ubuntu 12.04 com a Virtualbox a correr o Windows 7.

No seu estado actual, a calculadora não possui uma distribuição para macintosh devido à falta de acesso a uma máquina com essa plataforma. Esta limitação será corrigida num futuro próximo!
